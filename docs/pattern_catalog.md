# Emergent Pattern Catalog

Living catalog of emergent behavioral competencies observed in minimal agent-based systems.

## Catalog Entries

### P1: Spontaneous Aggregation
- **Description**: Agents with similar properties cluster spatially without explicit grouping instructions.
- **Canonical model**: Zhang et al. cell-view sorting (algotype clustering)
- **Also observed in**: Schelling segregation
- **Key metric**: Spatial autocorrelation / cluster size distribution
- **Status**: Planned

### P2: Delayed Gratification
- **Description**: Agents accept short-term cost (moving away from goal) to achieve long-term benefit (global sorting).
- **Canonical model**: Zhang et al. cell-view sorting
- **Key metric**: DG index (fraction of steps where agent moves away from final position)
- **Status**: Planned

### P3: Self-Organized Criticality
- **Description**: System self-tunes to a critical state with power-law distributed avalanches.
- **Canonical model**: Bak-Tang-Wiesenfeld sandpile
- **Key metric**: Power-law exponent of avalanche size distribution
- **Status**: Planned

### P4: Collective Motion / Alignment
- **Description**: Agents achieve global directional alignment through local interactions only.
- **Canonical model**: Reynolds Boids
- **Key metric**: Order parameter (mean alignment), polarization
- **Status**: Planned

### P5: Edge-of-Chaos Computation
- **Description**: System operates at boundary between ordered and chaotic regimes, maximizing computational capacity.
- **Canonical model**: Conway's Game of Life (select rule variants)
- **Key metric**: Lempel-Ziv complexity, lambda parameter
- **Status**: Planned

### P6: Associative Memory
- **Description**: System stores and retrieves patterns through attractor dynamics.
- **Canonical model**: Boolean gene regulatory networks
- **Key metric**: Pattern completion accuracy, basin size
- **Status**: Planned

### P7: Homeostatic Regulation
- **Description**: System maintains internal variable near set-point despite external perturbation.
- **Canonical model**: Various (adversarial perturbation protocol)
- **Key metric**: Recovery time, deviation integral
- **Status**: Planned

### P8: Stochastic Resonance
- **Description**: Noise improves system performance at an intermediate optimal level.
- **Canonical model**: Various
- **Key metric**: Performance vs. noise curve, optimal noise level
- **Status**: Planned

### P9: Equifinality / Robustness
- **Description**: System converges to same final state from diverse initial conditions.
- **Canonical model**: Various
- **Key metric**: Basin of attraction size, convergence variance
- **Status**: Planned
