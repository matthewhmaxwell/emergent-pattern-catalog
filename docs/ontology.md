# Pattern Ontology v0.4

Structural dimensions for classifying emergent behavioral competencies.

## Ontological Dimensions

### 1. Spatial Scale
- **Local**: Pattern involves individual agent behavior
- **Mesoscale**: Pattern involves clusters or neighborhoods
- **Global**: Pattern involves system-wide coordination

### 2. Temporal Character
- **Steady-state attractor**: Persists at equilibrium / attractor
- **Transient**: Emerges and disappears during dynamics
- **Oscillatory**: Sustained periodic dynamics
- **Propagating**: Traveling structures or wavefronts

### 3. Interaction Type
- **Direct**: Agent-to-agent interaction
- **Indirect-stigmergic**: Through environment modification
- **Global field**: Agents respond to a shared external signal
- **None (entropy-driven)**: No interaction; behavior from excluded volume or entropy

### 4. Interaction Substrate
- **Continuous space**: Agents move freely in Euclidean space
- **Discrete lattice**: Grid-based agents (CA, Schelling, etc.)
- **Fixed network**: Agents on static graph topology
- **Evolving network**: Graph topology changes during dynamics
- **Well-mixed compartment**: No spatial structure; random encounters

### 5. Agent Homogeneity
- **Identical**: All agents share the same rules and initial capabilities
- **Heterogeneous**: Agents differ in type, parameters, or information

### 6. Goal Structure
- **Goalless**: No objective function or preference
- **Implicit**: Behavior consistent with goal-seeking without explicit objective
- **Explicit**: Agents have programmed objectives

### 7. Feedback Structure
- **Positive**: Amplifying feedback loops
- **Negative**: Dampening / corrective feedback
- **Mixed**: Both positive and negative feedback present
- **Nontransitive**: Cyclic dominance (A > B > C > A)

### 8. Memory
- **Memoryless**: No state retention
- **Local state**: Agent carries internal state
- **Environmental trace**: Information persists in the environment

### 9. Conflict Structure
- **Cooperative**: Agents benefit from mutual interaction
- **Competitive**: Zero-sum or adversarial
- **Mixed-motive**: Cooperation and competition coexist (e.g., Prisoner's Dilemma)
- **Nontransitive**: Cyclic dominance relationships

### 10. External Driving
- **Autonomous**: Fully endogenous self-organization
- **Externally forced**: Sustained external perturbation
- **Periodically driven**: Rhythmic external input
- **Field-coupled**: Agents respond to an external gradient or field

### 11. Update Mode
- **Motion-transport**: Agents relocate in space
- **State transition**: Agents change internal state on fixed substrate
- **Reproduction-selection**: Strategies replicate differentially
- **Resource exchange**: Conserved quantity transfers between agents

## Dimensional Gap Analysis

Regions of the dimensional space that remain underexplored in the current catalog:

- **Heterogeneous agents + global field + no agent-agent interaction**: Only the
  deferred Brazil Nut Effect occupies this region
- **Evolving network substrate**: No current Layer 1 entries; network co-evolution
  with behavior is a known research frontier
- **Externally forced + nontransitive feedback**: No entries combine external
  driving with cyclic dominance
- **Resource exchange + heterogeneous agents**: P28 (wealth condensation) covers
  identical agents; heterogeneous-endowment variants may produce distinct behaviors
