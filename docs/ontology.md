# Pattern Ontology

Structural dimensions for classifying emergent behavioral competencies.

## Dimensions

### 1. Spatial Scale
- **Local**: Pattern involves individual agent behavior
- **Mesoscale**: Pattern involves clusters or neighborhoods
- **Global**: Pattern involves system-wide coordination

### 2. Temporal Scale
- **Instantaneous**: Observable in single snapshot
- **Transient**: Emerges and disappears during dynamics
- **Steady-state**: Persists at equilibrium / attractor

### 3. Cognitive Analogue
- **Perception**: Sensitivity to context (P2)
- **Memory**: State retention (P6)
- **Decision-making**: Trade-off navigation (P2)
- **Homeostasis**: Regulatory control (P7)
- **Collective intelligence**: Group-level coordination (P1, P4)

### 4. Mechanism Class
- **Self-organization**: Bottom-up pattern formation (P1, P3, P4)
- **Attractor dynamics**: Convergence to fixed points (P6, P9)
- **Criticality**: Phase transition phenomena (P3, P5)
- **Noise exploitation**: Constructive use of randomness (P8)

## Gap Analysis

Dimensions not yet covered by catalog entries:
- **Competition / cooperation dynamics**
- **Hierarchical organization emergence**
- **Communication / signaling**
- **Adaptive rule modification**
- **Division of labor**
